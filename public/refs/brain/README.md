# DBD capture images

Images pulled from Apple Notes "RUNI brain" captures land here, named after the
capture id (e.g. `2026-06-07-linear-empty-state-1.png`). They're referenced from
`src/content/captures.ts` as `/refs/brain/<file>` and shown on the `/dbd` page.

The `brain-dump` skill writes these — don't add files here by hand.
